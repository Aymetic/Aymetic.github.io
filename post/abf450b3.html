常用三角函数备忘录

发表于2021-03-26|Math
|字数总计:1.2k|阅读时长:7分钟|阅读量:|评论数:

最近复习考研数学中，经常遇到三角函数的换算，有难有易，为了备忘，还是将其进行归纳总结一下吧，到时候查阅也方便一些。

基本公式

sec⁡α=1cos⁡α\sec \alpha = \frac{1}{\cos \alpha}secα=cosα1​

csc⁡α=1sin⁡α\csc \alpha = \frac{1}{\sin\alpha}cscα=sinα1​

和差角公式

cos⁡(α+β)=cos⁡α⋅cos⁡β−sin⁡α⋅sin⁡β\cos(α+β)=\cosα·\cosβ-\sinα·\sinβcos(α+β)=cosα⋅cosβ−sinα⋅sinβ

cos⁡(α−β)=cos⁡α⋅cos⁡β+sin⁡α⋅sin⁡β\cos(α-β)=\cosα·\cosβ+\sinα·\sinβcos(α−β)=cosα⋅cosβ+sinα⋅sinβ

sin⁡(α±β)=sin⁡α⋅cos⁡β±cos⁡α⋅sin⁡β\sin(α±β)=\sinα·\cosβ±\cosα·\sinβsin(α±β)=sinα⋅cosβ±cosα⋅sinβ

tan⁡(α+β)=tan⁡α+tan⁡β1−tan⁡α⋅tan⁡β\tan(α+β)=\frac{\tanα+\tanβ}{1-\tanα·\tanβ}tan(α+β)=1−tanα⋅tanβtanα+tanβ​

tan⁡(α−β)=tan⁡α−tan⁡β(1+tan⁡α⋅tan⁡β)\tan(α-β)=\frac{\tanα-\tanβ}{(1+\tanα·\tanβ)}tan(α−β)=(1+tanα⋅tanβ)tanα−tanβ​

半角公式

tan⁡α2=1−cos⁡αsin⁡α=sin⁡α1+cos⁡α\tan \frac{\alpha }{2}=\frac{1-\cos \alpha }{\sin \alpha }=\frac{\sin \alpha }{1+\cos \alpha }tan2α​=sinα1−cosα​=1+cosαsinα​

cot⁡α2=sin⁡α1−cos⁡α=1+cos⁡αsin⁡α\cot \frac{\alpha }{2}=\frac {\sin \alpha }{1-\cos \alpha }=\frac {1+\cos \alpha }{\sin \alpha }cot2α​=1−cosαsinα​=sinα1+cosα​

sin⁡2α2=1−cos⁡α2\sin^2\frac{\alpha}{2}=\frac {1-\cos \alpha}{2}sin22α​=21−cosα​

cos⁡2α2=1+cos⁡α2\cos^2\frac{\alpha}{2}=\frac{1+\cos \alpha}{2}cos22α​=21+cosα​

tan⁡α2=1−cos⁡αsin⁡α=sin⁡α1+cos⁡α\tan\frac{\alpha}{2}=\frac{1-\cos \alpha}{\sin \alpha}=\frac{\sin \alpha}{1+\cos\alpha}tan2α​=sinα1−cosα​=1+cosαsinα​

二倍角公式

sin⁡2α=2sin⁡αcos⁡α\sin {2\alpha }=2\sin\alpha\cos\alphasin2α=2sinαcosα

cos⁡2α=cos⁡α2−sin⁡α2=1−2sin⁡α2=2cos⁡α2−1\cos 2\alpha =\cos \alpha ^2-\sin \alpha ^2=1-2\sin \alpha ^2=2\cos \alpha ^2-1cos2α=cosα2−sinα2=1−2sinα2=2cosα2−1

tan⁡2α=2tan⁡α1−tan⁡α2\tan 2\alpha =\frac{2\tan \alpha }{1-\tan \alpha ^2}tan2α=1−tanα22tanα​

三倍角公式

sin⁡3α=4sin⁡α⋅sin⁡(π3+α)⋅sin⁡(π3−α)\sin 3\alpha=4\sin\alpha·\sin(\frac{\pi}{3}+α)·\sin(\frac{\pi}{3}-α)sin3α=4sinα⋅sin(3π​+α)⋅sin(3π​−α)

cos⁡3α=4cos⁡α⋅cos⁡(π3+α)⋅cos(π3−α)\cos 3α=4\cosα·\cos(\frac{\pi}{3}+α)·cos(\frac{\pi}{3}-α)cos3α=4cosα⋅cos(3π​+α)⋅cos(3π​−α)

tan⁡3α=tan⁡α⋅tan⁡(π3+α)⋅tan(π3−α)\tan3\alpha = \tan \alpha · \tan(\frac{\pi}{3}+\alpha)· tan(\frac{\pi}{3}-\alpha)tan3α=tanα⋅tan(3π​+α)⋅tan(3π​−α)

辅助角公式

Asin⁡α+Bcos⁡α=A2+B2sin⁡(α+t)A\sinα+B\cosα=\sqrt{A^2+B^2}\sin(α+t)Asinα+Bcosα=A2+B2​sin(α+t)

其中

sin⁡t=B/(A2+B2)(1/2)\sin t=B/(A^2+B^2)^(1/2)sint=B/(A2+B2)(1/2)

cos⁡t=AA2+B2\cos t=\frac{A}{\sqrt{A^2+B^2}}cost=A2+B2​A​

tan⁡t=BA\tan t=\frac{B}{A}tant=AB​

Asin⁡α+Bcos⁡α=A2+B2cos⁡(α−t)，其中tan⁡t=ABA\sinα+B\cosα=\sqrt{A^2+B^2}\cos(α-t)，其中\tan t=\frac{A}{B}Asinα+Bcosα=A2+B2​cos(α−t)，其中tant=BA​

降幂公式

sin⁡2α=1−cos⁡2α2=versin2α2\sin^2α=\frac{1-\cos2α}{2}=\frac{versin2α}{2}sin2α=21−cos2α​=2versin2α​

cos⁡2α=1+cos⁡2α2=covers2α2\cos^2α=\frac{1+\cos2α}{2}=\frac{covers2α}{2}cos2α=21+cos2α​=2covers2α​

tan⁡2α=1−cos⁡2α1+cos⁡2α\tan^2α=\frac{1-\cos2α}{1+\cos2α}tan2α=1+cos2α1−cos2α​

和差化积

sin⁡θ+sin⁡φ=2sin⁡[θ+φ2]cos⁡[θ−φ2]\sinθ+\sinφ = 2 \sin[\frac{θ+φ}{2}] \cos[\frac{θ-φ}{2}]sinθ+sinφ=2sin[2θ+φ​]cos[2θ−φ​]

sin⁡θ−sin⁡φ=2cos⁡[θ+φ2]sin⁡[θ−φ2]\sinθ-\sinφ = 2 \cos[\frac{θ+φ}{2}] \sin[\frac{θ-φ}{2}]sinθ−sinφ=2cos[2θ+φ​]sin[2θ−φ​]

cos⁡θ+cos⁡φ=2cos⁡[θ+φ2]cos⁡[θ−φ2]\cosθ+\cosφ = 2\cos[\frac{θ+φ}{2}] \cos[\frac{θ-φ}{2}]cosθ+cosφ=2cos[2θ+φ​]cos[2θ−φ​]

cos⁡θ−cos⁡φ=−2sin⁡[θ+φ2]sin⁡[θ−φ2]\cosθ-\cosφ = -2\sin[\frac{θ+φ}{2}] \sin[\frac{θ-φ}{2}]cosθ−cosφ=−2sin[2θ+φ​]sin[2θ−φ​]

tan⁡A+tan⁡B=sin⁡(A+B)cos⁡Acos⁡B=tan⁡(A+B)(1−tan⁡Atan⁡B)\tan A+\tan B=\frac{\sin(A+B)}{\cos A\cos B}=\tan(A+B)(1-\tan A\tan B)tanA+tanB=cosAcosBsin(A+B)​=tan(A+B)(1−tanAtanB)

tan⁡A−tan⁡B=sin⁡(A−B)cos⁡Acos⁡B=tan⁡(A−B)(1+tan⁡Atan⁡B)\tan A-\tan B=\frac{\sin(A-B)}{\cos A\cos B}=\tan(A-B)(1+\tan A\tan B)tanA−tanB=cosAcosBsin(A−B)​=tan(A−B)(1+tanAtanB)

积化和差

sin⁡αsin⁡β=cos⁡(α−β)−cos⁡(α+β)2\sinα\sinβ = \frac{\cos(α-β)-\cos(α+β)}{2}sinαsinβ=2cos(α−β)−cos(α+β)​

cos⁡αcos⁡β=cos⁡(α+β)+cos⁡(α−β)2\cosα\cosβ = \frac{\cos(α+β)+\cos(α-β)}{2}cosαcosβ=2cos(α+β)+cos(α−β)​

sin⁡αcos⁡β=sin⁡(α+β)+sin⁡(α−β)2\sinα\cosβ = \frac{\sin(α+β)+\sin(α-β)}{2}sinαcosβ=2sin(α+β)+sin(α−β)​

cos⁡αsin⁡β=sin⁡(α+β)−sin⁡(α−β)2\cosα\sinβ = \frac{\sin(α+β)-\sin(α-β)}{2}cosαsinβ=2sin(α+β)−sin(α−β)​

万能公式

万能公式是将sin⁡x,cos⁡x\sin x,\cos xsinx,cosx和tan⁡x\tan xtanx均用tan⁡π2\tan \frac{\pi}{2}tan2π​表示。由于后者的值域为整个实数区间，因此方便考察许多性质。

首先我们知道，tan⁡x\tan xtanx的万能公式就是其二倍角公式。我们试着推导一下余弦函数的万能公式。

cos⁡x=1−tan⁡2x21+tan⁡2x2\cos x = \frac{1-\tan^2\frac{x}{2}}{1+\tan^2\frac{x}{2}}cosx=1+tan22x​1−tan22x​​

sin⁡x=cos⁡xtan⁡x=1−tan⁡2x21+tan⁡2x2⋅2tan⁡x21−tan⁡2x2=2tan⁡x1−tan⁡2x\sin x = \cos x\tan x=\frac{1-\tan^2\frac{x}{2}}{1+\tan^2\frac{x}{2}}·\frac{2\tan\frac{x}{2}}{1-\tan^2\frac{x}{2}}=\frac{2\tan x}{1-\tan^2x}sinx=cosxtanx=1+tan22x​1−tan22x​​⋅1−tan22x​2tan2x​​=1−tan2x2tanx​
